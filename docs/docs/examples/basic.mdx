### Preserve order
If you are interessed to preserve order in selection you can set the property **preserve-order** to ***false***.

        selected='["IT"]'

        options='[{"value":"IT", "label":"Italy"},{"value":"FR", "label":"France"},{"value":"ES", "label":"Spain"},{"value":"DE","label":"Germany"}]'>

### Disable multiselection
You can use cn-multiselect also for generate a select input or radio input.\
If **multiple** property is ***false*** then select/deselect all actions are not available.

<Preview component={<cn-multiselect
        selected='["IT"]'
        multiple="false"
        title="Select country"
        options='[{"value":"IT", "label":"Italy"},{"value":"FR", "label":"France"},{"value":"ES", "label":"Spain"},{"value":"DE","label":"Germany"}]'>
</cn-multiselect>}
      source={`<cn-multiselect
        title="Select country"
        multiple="false"
        selected='["IT"]'
        options='[{"value":"IT", "label":"Italy"},{"value":"FR", "label":"France"},{"value":"ES", "label":"Spain"}]'>
</cn-multiselect>`}/>
